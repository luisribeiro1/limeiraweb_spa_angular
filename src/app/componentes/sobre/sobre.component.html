O projeto Temperatura em Limeira surgiu com o propósito de disponibilizar, de forma amigável e compreensível, os dados gerados pela estação meteorológica da Unicamp, que está posicionada fisicamente no campus I - Faculdade de Tecnologia. Os dados "sem tratamento" gerado pela estação podem ser acessados aqui: https://www3.ft.unicamp.br/meteorologia/downld08.txt

Observação: Os dados apresentados neste site devem ser usados com cuidado, uma vez que a estação metereológica fica inoperante por alguns períodos, resultando em um apagão no fornecimento dos dados.

O projeto foi criado por Luis Ribeiro, analista de sistemas e docente na área de tecnologia da informação do Senac Limeira, está disponível há vários anos, agora em sua segunda versão.

Tecnologias usadas na primeira versão

  • HTML5 e CSS3
  • Linguagem PHP, banco de dados MySQL e linguagem SQL
  • jQuery - framework Javascript
  • Bootstrap - framework CSS e ícones
  • MDBootstrap - extensão do Bootstrap
  • CloudFlare - segurança, DNS e outros
  • Let's Encrypt - Certificado SSL/TLS

Tecnologias usadas na segunda versão (atual)

  • Front-end: HTML5 e CSS3 (SCSS), Angular e Bootstrap
  • Back-end: Java com Spring Boot. Banco MySQL
  • CloudFlare - segurança, DNS e outros
  • Let's Encrypt - Certificado SSL/TLS

Este site usa o conceito de design minimalista.
